About the Project

This e-learning app was developed as part of my dissertation project, utilizing Flutter and Dart.js to create a cross-platform learning solution. The app allows for 'teachers' and 'students' to interact within a fully-featured educational platform focused on cybersecurity education, the platform is designed with an educational setting in mind but has the scope to be utilised by SMOs and those without access to paid cyber security resources .

'Teachers' can sign up and use a built-in lesson creation tool to design and manage courses on cybersecurity, including video lessons, text-based content, and quizzes. 'Students' can view these lessons, track their progress, and engage with the material across multiple devices, whether on mobile or desktop.

The app also includes an admin panel for managing users and content, while leveraging Firebase for secure user authentication (via Google SSO) and cloud storage of multimedia files. Firestore is used for real-time database management, allowing for seamless tracking of user progress and content updates.

Key features of the app include a course search page, a save/bookmark feature, and a user dashboard that shows progress metrics and achievements. The app is designed to be secure, with Google SSO for authentication and robust data protection measures in place.

Tools Used:

  • Flutter
  • Dart.js
  • Firebase Authentication
  • Firestore Database
  • Firebase Cloud Storage
  • Google SSO

What I learned:

  • Building a cross-platform app using Flutter and Dart.js for a responsive, portable solution.
  • Creating dynamic and interactive content with a lesson creation tool for teachers.
  • Integrating Google SSO for user authentication, ensuring secure sign-ins and data protection.
  • Using Firebase Firestore to manage real-time data and track user progress.
  • Implementing cloud storage with Firebase to handle multimedia content like videos and Lessons.
  • Building a user dashboard to provide real-time metrics and insights into learning progress.
  • Designing and implementing a course search page to help users find relevant lessons quickly.
  • Ensuring security with best practices such as encrypted user authentication and secure file storage.
