About the Project

"Gels by G" is a mobile gel nail technician service based in Kent, UK. The client wanted a website that would allow customers to view the services offered, view previous work, book appointments, and contact the business.

The website was designed using Webflow, a no-code platform that allows for the creation of responsive websites without the need for coding, this allowed for the website to be initially designed and created by me but all future content updates are able to be performed by the website owner with little to no previous knowledge required.

Tools Used:

  • Webflow
  • Adobe Photoshop
  • Netlify

What I learned:

  • How to design and develop a website using Webflow
  • Integrating typeforms for business contact
  • Using Netlify to host from a webflow deploy
