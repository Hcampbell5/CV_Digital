About the Project

This project is a custom Android game built using Java and Gradle. The game features a dynamic, branching storyline where players make choices that influence the narrative, akin to choose-your-own-adventure books or the interactive episode Black Mirror: Bandersnatch.

The storylines are defined in a structured .csv file, which the game reads to present choices and determine outcomes. By making different choices, players can unlock alternative story paths and endings, creating a deeply engaging experience.

I designed the game's user interface in Photoshop, ensuring a visually appealing and user-friendly experience. The storyline, crafted entirely by me, emphasizes creativity and complex decision trees to deliver an immersive narrative experience.

Tools Used:

  • Java
  • Gradle
  • Photoshop
  • Android studio

What I learned:

  • How to design front and backend for android applications
  • Creating a fully-featured Android application, including splash page
  • Using Gradle to automate the building, compiling and package linking
  • Building a server in java to connect the application to the database storing storyline options
  • Using Android Studio and Photoshop to create interesting and responsive UI
  • Creating a dynamic, branching narrative structure for an interactive game
